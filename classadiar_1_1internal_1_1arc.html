Detailed Description

An arc from some source to a target.

An arc contains a value for the source and one for the target. Notice, that we don't specifically have a member for is_high. The reason for this choice is that the C++ compiler makes everything word-aligned. That means with an explicit is_high member it would take up 3 x 64 bits rather than only 2 x 64 bits.

Remarks
If source is flagged, then this is a high arc rather than a low arc.
source may be NIL
