Detailed Description

template<uint8_t CARDINALITY, uint8_t NODE_CARRY_SIZE = 0u, uint8_t INPUTS = CARDINALITY>
class adiar::internal::request< CARDINALITY, NODE_CARRY_SIZE, INPUTS >

Request struct for the top-down sweep time-forwarding algorithms.

Template Parameters
cardinalityThe cardinality of the algorithm, e.g. it is 1 for bdd_restrict, 2 for bdd_apply and 3 for bdd_ite.
nodes_carriedNumber of children being forwarded with the request. This is used when cardinality is greater than 1 and a per-level priority queue forwards the children of target.fst() to target.snd() and so on.
