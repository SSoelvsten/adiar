Variable Documentation

◆ mode

memory_mode adiar::memory::mode

The current memory mode (default: AUTO).

If you want to force Adiar to a specific memory mode then you should set the global variable adiar::memory::mode to one of the above three values. For example, one can force Adiar always use internal memory with the following piece of code.

adiar::memory::mode = adiar::memory::INTERNAL
Warning
Using INTERNAL may lead to crashes if an input or output is too large!
