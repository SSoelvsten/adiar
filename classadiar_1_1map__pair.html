Detailed Description

template<typename key_type, typename value_enum = boolean>
class adiar::map_pair< key_type, value_enum >

A (x,v) tuple representing the single association \( x \mapsto v \) where \(v\) is a value of enum type.

Template Parameters
key_typeThe (integral) type for the map's key.
value_enumAn enum type that has the two values 'False' with the integral value '0' and 'True' with the integral value '1', e.g. bool_enum.
