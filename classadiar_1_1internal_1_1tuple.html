Detailed Description

template<typename T, uint8_t Cardinality = 2, bool IsSorted = false>
class adiar::internal::tuple< T, Cardinality, IsSorted >

Tuple holding elements and providing an ordered access.

Template Parameters
value_tThe type of elements within the tuple. This part should have a partial ordering '<'.
cardinalityThe number of elements in the tuple (default: 2)
is_sortedWhether elements are given in sorted order (default: false).
