Detailed Description

template<typename T, typename Comp = no_ordering<T>>
class adiar::simple_file_writer< T, Comp >

Write-only access to a simple file including a consistency check on the given input.

Parameters
TType of the file's content
CompThe logic to check for consistency (usually a comparator)

The consistency check verifies, whether something is allowed to come after something else. In all our current use-cases, the check induces a total ordering.
